# SimpleBlogSystem
A simple blog system

### Pre-requisition
Golang >= 1.15.5
MongoDB >= 4.4

### Build
To build, please run the following command:
`
make build
`

### Program Execution
`
start.sh
`

### Unit Testing
`
go test
`

### MongoDB
MongoDB data files will be created in data folder of the project.

- Database -> sbs
- Collection -> articles
